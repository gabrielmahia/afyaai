# AfyaAI — Kenya Health Assistant

AI-powered health guidance for Kenya — maternal health, NHIF/SHA navigation, symptoms checker.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://afyaai.streamlit.app)

## Features

- 🤰 **Maternal Health** — Third trimester guide, nutrition, warning signs, birth preparedness
- 💊 **General Health** — Kenya-specific guidance on malaria, diabetes, TB, hypertension
- 🏥 **NHIF/SHA Guide** — Coverage, registration, maternity benefits, Linda Mama programme
- 🧪 **Symptoms Checker** — AI-powered triage with Kenya health system context
- 💬 **Ask AfyaAI** — Free-form health Q&A

## Deployment

[![Deploy on Streamlit Cloud](https://streamlit.io/images/brand/streamlit-logo-primary-colormark-darktext.png)](https://share.streamlit.io)
